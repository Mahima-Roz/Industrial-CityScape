This project presents a 2D animated urban-industrial environment built using OpenGL and GLUT in C++. The scene represents a busy industrial city located near the ocean where transportation, industry, and natural elements interact in a dynamic environment.

The project integrates multiple animated objects, environmental effects, and user interactions to simulate a lively industrial area. Vehicles move on the road with rotating wheels, ships travel across the ocean, windmills rotate beside buildings, and bubbles rise from a nuclear plant. A plane flies in the sky while a jellyfish moves inside the ocean.

A major highlight of the project is the use of texture mapping, which significantly improves visual realism. The sky, sunset, moon, and ocean surface are texture-based, creating a more immersive graphical environment. 

🛠 Technologies Used
Technology	Purpose
C++	Core programming language
OpenGL	2D graphics rendering
GLUT	Window management and interaction
Texture Mapping	Realistic sky, ocean, and sunset effects
Animation Functions	Object movement and simulation

⭐ Key Features
🌇 Texture-Based Visual Environment

Texture mapping for realistic sky

Sunset texture during day mode

Moon and dark sky texture during night mode

Tortoise-textured ocean surface

🚗 Animated Transportation System

Moving Car, Truck, Lorry, and Forklift

Rotating vehicle wheels

Vehicles can be paused or resumed via keyboard

🚢 Ocean Transportation

Moving Ship, Boat, and Submarine

Smooth left-right movement animation

Objects can be controlled individually

🌬 Industrial Animation

Rotating windmills

Bubbles emitted from nuclear plant

Signal tower and industrial buildings

🌧 Environmental Effects

Rain animation

Flying plane

Moving jellyfish inside ocean

🖱 User Interaction

Mouse control for switching Day/Night

Keyboard control for animation toggling

Scene switching between Scene 1 and Scene 2
